# Trading Dashboard Frontend

A modern real-time trading dashboard built with **React + TypeScript + Vite**.

---

## Project Overview

This frontend application displays live data such as tickers, charts, and price updates.
It is designed for performance, responsiveness, and a clean UI.

---

## Features

- Fast Vite-powered React app
- Real-time charting with Recharts
- Ready for authentication integration
- React Router navigation
- TypeScript support for scalability

---

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router DOM
- Recharts
- ESLint

---

## Installation

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

## Notes

- Built using Vite for fast HMR
- React Router v7 for routing
- Recharts used for visualization
